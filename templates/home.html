Instructions

Basic usage

{{ .Host }}/640x480

To obtain a place holder image, add the desired resolution in a "width x height" format to the address bar, as illustrated above.

Shortcut for square images

{{ .Host }}/640

The height attribute can be omitted for square placeholder images.

Text customization

{{ .Host }}/640x480?text=Lorem+ipsum+sur+amet

By default, the text generated by a placeholder image is its' resolution. Adding a text attribute on a query string will override this behavior.

Color customization

{{ .Host }}/640x480/000/FFFFFF

Colors for the placeholder images can be edited by adding color hexcodes after the image resolution. The first hexcode represents the background color, while the second one represents the text color. (Keep in mind that changing the text color cannot be done in the absence of a background color.)

Image formats

{{ .Host }}/640x480.jpg

placeholder-server supports the following formats for image generation: GIF, JPEG, and PNG(default). Specify your desired format by adding its' file extension to the end of the image's URL.
